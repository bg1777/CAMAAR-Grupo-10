    class Form

    Representa um formulário publicado para uma turma específica. Baseado em um FormTemplate, possui status (rascunho, publicado, fechado) e armazena as respostas dos alunos através de FormResponse.

def completed_responses
  form_responses.map(&:user)
end

    Retorna todos os alunos que já responderam o formulário.

    Efeitos Colaterais¶ ↑

def pending_responses
  klass.students - form_responses.map(&:user)
end
